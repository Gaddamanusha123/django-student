I developed a Student Management System using Django.
In this project, I created a system where we can add, update, and manage student details like name, age, course, and roll number through the Django Admin panel.

What makes your project special 

I also implemented role-based access control in the admin panel.

Admin user can:
View all data
Add, edit, and delete students
Access all fields
Normal user can:
See only limited fields (like name and course)
Edit only allowed fields
Cannot delete data
